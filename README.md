# 🌍 Флаги Мира — Угадай Страну

Браузерная игра о странах мира с тремя режимами. Никакого бэкенда — чистый HTML, CSS и JavaScript, задеплоено через GitHub Pages.

**[▶ Играть →](https://ithemaybe.github.io/game/)**

---

## Режимы игры

### 🏳️ V1 — Угадай по флагу
Показывается флаг страны и 4 варианта ответа. Нужно выбрать правильный. **60 секунд** — сколько успеешь?

- Управление мышью, тапом или с клавиатуры (1–4 / стрелки)
- Счётчик правильных и ошибочных ответов в реальном времени
- Круговой таймер
- Экран результатов с процентом точности

### 🗺️ V2 — Назови страны

Во втором режиме можно выбрать один из двух вариантов игры:

#### ⌨️ Текстовый режим
Классический режим с полем ввода. **10 минут** — вспомни как можно больше стран мира.

- Принимаются названия на русском языке
- Страны вводятся вручную через поле ввода
- Повторы и несуществующие страны не засчитываются
- Принятые страны отображаются тегами в реальном времени
- Прогресс-бар показывает результат из 195 стран

#### 🎙️ Голосовой режим
Отдельный режим с голосовым вводом. Игрок разрешает доступ к микрофону и произносит страны вслух.

- Браузер запрашивает доступ к микрофону
- Можно произносить сразу несколько стран подряд
- Каждая распознанная страна засчитывается отдельно
- Повторы и несуществующие страны не засчитываются
- Принятые страны отображаются тегами в реальном времени
- Для текстового и голосового режима сохраняются отдельные личные рекорды

В режимах результат можно скопировать как ссылку и кинуть вызов другу.


### 🏳️ V3 — Угадай флаг без таймера

Показывается один флаг, а игрок вводит название страны вручную. Ограничения по времени нет: можно завершить игру в любой момент или пройти весь набор флагов.

- Свободный текстовый ответ вместо вариантов
- Поддержка распространённых альтернативных названий стран
- Счётчик правильных ответов и ошибок
- Прогресс по всему набору стран
- Кнопка досрочного завершения и экран результатов
- Отдельный личный рекорд

## Структура проекта

```
game/
├── index.html        # V1 — угадай страну по флагу
├── game.js           # Логика V1
├── v2.html           # V2 — назови страны по памяти
├── v2.js             # Логика V2
├── v3.html           # V3 — угадай флаг без таймера
├── v3.js             # Логика V3
├── style-v3.css      # Стили V3
├── flags.json        # Общий список стран и флагов
├── style.css         # Стили для V1
├── swimming-cat.js   # Плавающий котик
├── result.html       # Страница результата (шэринг)
└── 404.html          # Кастомная страница ошибки
```

---

## Технологии

- Чистый **HTML / CSS / JavaScript** — без фреймворков и зависимостей
- Флаги — публичный CDN [flagcdn.com](https://flagcdn.com) или emoji-флаги в режиме оптимизации
- Шрифты — Unbounded + Inter (Google Fonts)

---

## Общие возможности

- Переключатель V1 / V2 / V3 в верхней части каждой игровой страницы
- Опциональная оптимизация загрузки флагов
- Плавающий котик на всех режимах
- Хранение личных рекордов в `localStorage`

### Управление режимами и пропуск

Переключатель V1 / V2 / V3 поддерживает обычное нажатие и жест: зажмите любую плашку, проведите пальцем или мышью к нужному режиму и отпустите. В V3 доступна кнопка «Пропустить флаг» — пропуск переводит к следующей стране, не добавляя ошибку.
